# Astra Quick Start Guide

Get Astra running in under 5 minutes.

---

## Option 1: Docker (Recommended)

```bash
# 1. Clone the repository
git clone https://github.com/astra-ai/astra.git
cd astra

# 2. Copy environment file and add your API keys
cp .env.example .env
# Edit .env - add at least one LLM API key (OPENAI_API_KEY, ANTHROPIC_API_KEY, or GOOGLE_AI_API_KEY)

# 3. Start everything
docker-compose up -d

# 4. Open in browser
open http://localhost:3000
```

**That's it!** You now have:
- Web Dashboard at http://localhost:3000
- API at http://localhost:3000/api/v1
- API Docs at http://localhost:3000/api/v1/docs

---

## Option 2: Local Development

```bash
# 1. Clone and enter directory
git clone https://github.com/astra-ai/astra.git
cd astra

# 2. Run setup script
chmod +x setup.sh
./setup.sh

# 3. Edit .env with your API keys
# nano .env

# 4. Start the server
npm run start:api
```

## First Steps After Installation

### 1. Add Your API Keys

Astra uses a BYOK (Bring Your Own Keys) model. Add at least one LLM provider key:

**Via Dashboard:**
1. Open http://localhost:3000
2. Login with OAuth (Google, GitHub, or Apple)
3. Go to "API Keys" in the sidebar
4. Add your OpenAI/Anthropic/Google key

**Via API:**
```bash
curl -X POST http://localhost:3000/api/v1/keys \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "openai",
    "key": "sk-...",
    "name": "My OpenAI Key"
  }'
```

### 2. Start Chatting

**Via Dashboard:**
1. Click "Chat" in the sidebar
2. Type your message and press Enter

**Via API:**
```bash
curl -X POST http://localhost:3000/api/v1/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello, Astra!"}'
```

### 3. Connect Messaging Platforms (Optional)

**WhatsApp:**
1. Go to "Platforms" in the dashboard
2. Click WhatsApp
3. Scan the QR code with your phone

**Telegram:**
1. Create a bot with @BotFather
2. Add the token to your .env file
3. Restart the server

## Environment Variables

The minimum required configuration:

```bash
# At least one of these LLM keys:
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
GOOGLE_AI_API_KEY=...

# Security (auto-generated by setup.sh):
MASTER_ENCRYPTION_KEY=...
JWT_SECRET=...
```

See [.env.example](.env.example) for all options.

## Troubleshooting

### Port 3000 already in use
```bash
# Change the port in .env
PORT=3001
```

### Database connection failed
```bash
# Make sure PostgreSQL is running
docker-compose up -d db

# Or use local PostgreSQL
DATABASE_URL=postgres://user:pass@localhost:5432/astra
```

### Tests failing
```bash
# Run tests to see errors
npm test

# Check TypeScript compilation
npm run typecheck
```
